# LLProgramFramework
## 本项目是一个拿来即用的基础OC框架：


![LLProgramFramework.gif](https://github.com/liuniuliuniu/LLProgramFramework.Swift/blob/master/LLProgramFrameworkSwift.gif)
### 项目结构

* General 工具类综合
	* Category ------分类
	* UIKitAddtion ------UI控件分类
	* BaseClass  ------基类
* Resources ------资源文件
* Sections  ------模块组
	* Main   
	* Home
	* Me
* Service ------网络请求库
* Macro  ------pch以及全局变量
* Vendors  ------手动集成第三方库
	* [LLFMDB](https://github.com/liuniuliuniu/LLFMDB)  ------ 个人二次封装的FMDB 一键缓存的数据库
